# 瑞众保险 AI 投保原型

基于 HTML/CSS/JS 的交互式原型设计，包含完整的投保流程。

## 在线预览

直接打开 `prototype.html` 即可在浏览器中查看。

如需 GitHub Pages 在线访问，推送后前往 **Settings → Pages → Source: main branch** 即可获得在线链接。

## 功能页面

| 页面 | 说明 |
|------|------|
| P0 | 登录页 |
| P1 | 首页（产品展示、常见问题、产品详情） |
| P1-2 | 投保方案页 |
| P2 | 保障方案确认页 |
| P3 | 健康告知页 |
| P4 | 投保人信息填写 |
| P5 | 被保人信息 |
| P6 | 受益人信息 |
| P7 | 订单确认 |
| P8 | 投保确认（人脸识别） |
| P9 | 支付页 |
| P10 | 投保完成 |

## 技术说明

- 纯前端实现，无需后端服务
- 直接在浏览器打开 `prototype.html` 即可使用
- 支持手机模拟器预览效果
